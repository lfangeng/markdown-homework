# 欢迎使用 Markdown 在线编辑器

在左侧输入 Markdown，右侧会**实时预览**渲染结果。

## 功能亮点

- 左右分栏布局
- 实时 Markdown 渲染
- 支持代码块语法高亮
- 内置高亮兜底，不依赖单一 CDN
- 一键保存为 .md 文件
- 自动保存本地输入内容

## 代码示例

```javascript
function greet(name) {
  return `Hello, ${name}! 欢迎使用 Markdown 编辑器。`;
}

console.log(greet("Copilot"));
```
